# CodeSense

**CodeSense** is a powerful tool designed to analyze all the files inside a project folder, providing a comprehensive summary of the entire project structure. Whether you're dealing with a small script or a large-scale codebase, CodeSense helps you quickly understand the organization and functionality of your code.

## Features

- **Comprehensive Analysis**: Scans and analyzes all files in your project folder, generating a detailed summary.
- **Project Structure Overview**: Provides a brief, understandable summary of the entire project's structure.
- **Intelligent Processing**: Utilizes advanced language models to comprehend and summarize code effectively.

## Tools and Technologies

- **Langchain**: For natural language processing and understanding the code.
- **Groq**: Enables efficient computation and analysis, ensuring rapid processing.
- **Streamlit**: Powers the user interface, making CodeSense easy to use and interact with.
